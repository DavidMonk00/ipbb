# IPbus Firmware Builder

### The successor of dep_tree & ProjectManager

# Repository structure
 * my_area/
   * .ipbbarea
   * src/
     * repo_1/
     * repo_2/
     * repo_3/
   * build
     * proj_A/
       * .projarea
     * proj_B/
       * .projarea

# Example

Initialise a new working area: Creates the root and source folder

```
ipbb init myarea
```
